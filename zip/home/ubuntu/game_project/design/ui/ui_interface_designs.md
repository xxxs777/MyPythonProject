## 13. 交互流程图

### 主要导航流程
```
启动界面 → 主界面 → 
  ├─ 关卡地图 → 游戏界面 → 结算界面
  ├─ 庄园界面 → 建筑/装饰选择
  ├─ 萌宠界面
  ├─ 商店界面
  └─ 社交界面
```

### 游戏流程
```
关卡选择 → 游戏开始 → 
  ├─ 游戏胜利 → 结算界面 → 下一关/返回地图
  └─ 游戏失败 → 失败界面 → 重试/返回地图
```

### 庄园建设流程
```
庄园界面 → 建筑/装饰选择 → 
  ├─ 资源足够 → 放置建筑 → 建造完成
  └─ 资源不足 → 资源获取提示 → 商店/关卡
```

### 萌宠互动流程
```
萌宠界面 → 选择萌宠 → 
  ├─ 喂养 → 经验增加 → 可能升级
  ├─ 互动 → 亲密度增加 → 可能解锁技能
  ├─ 装扮 → 选择装饰 → 应用装饰
  └─ 出战 → 设为当前萌宠
```
